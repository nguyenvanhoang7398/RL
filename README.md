# Reinforcement Learning

## Getting started
Change your environment config in `parking/env.py`

Train DQN model
```
python dqn.py --train
```

Run reward shaping
```
python parking.py --rs
```

Train DAGGER
```
python parking.py --dagger
```

Test many
```
python parking.py --test-many
```

Test single
```
python parking.py --test-single
```
